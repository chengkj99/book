# NPM 命令

## nrm ls

NRM 是 NPM registry 管理工具，能够查看和切换当前使用的 registry

- nrm list // show all registries
- nrm help // show help
- nrm use cnpm // switch to cnpm
- nrm home // go to a registry home page

## npm init

创建一个 package.json 文件

- npm init --yes
- npm init
